
# Soundboard Pack Generator

A utility tool for creating custom soundboard ressource packs.

## Overview

Soundboard Pack Generator simplifies the process creating soundboard resource packs to be able to use them with the mod : [https://www.curseforge.com/minecraft/mc-mods/bzk-soundboard](https://www.curseforge.com/minecraft/mc-mods/bzk-soundboard).

## Features

- **Pack Creation**: Generate soundboard packs from audio files
- **Metadata Management**: Easily add titles to your soundboards
- **Multi-format Support**: Accepts .mp3, .wav, and .ogg files, converting them to .ogg
