Instructions

This is a tool that allows you to generate resource packs compatible with the mod: https://www.curseforge.com/minecraft/mc-mods/bzk-soundboard.

You need to select the audio files you want to include in your soundboard. Once selected, you can change the names of the sounds that will appear in the in-game menu.
Once finished, you can download the resource pack and place it in your usual resource packs folder.

You can choose files of type .mp3 .wav and .ogg they will all be automatically converted to .ogg

If you have any problems, feel free to open an issue on the github repository. https://github.com/BZKgenesis/soundboard-pack-generator/issues/new/choose
